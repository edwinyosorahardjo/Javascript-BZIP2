BZIP2 is a lossless data compression algorithm, it is available on a BSD based license.
It uses Burrows-Wheelers transform into Move To Front ending with Huffmans algorithm.
Similar to GZIP but slower. Has bigger compression rate in most cases.
Original BZIP2 was developed by Julian Seward and this implementation is not the original
